*The aim of the project is to device an algorithm for generating
*W-sets with fever elements. The files are either source codes 
* (c++,h files) or txt data files that are required to run.
*
*The format of an FSM sould be given as
*__________________________________________________________________________________________________________________________
*|Name id                                                                                                                 |
*|number of states number of transitions                                                                                  | 
*|source state (integer value) destination state (integer value), input (characrter value) and the output (integer value) |
*|________________________________________________________________________________________________________________________|
*
*The files were compiled using Microsoft Visual Studio .NET 2013. 
*For each folder you may create one project. 
*main.cpp is the main file for the execution. 

*Each programm receives fsm.txt file that contains 
*the targetted FSMs.
*The results are then given in  out.txt
